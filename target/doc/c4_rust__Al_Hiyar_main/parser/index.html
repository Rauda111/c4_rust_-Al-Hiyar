The parser module implements a recursive descent parser for a subset of C.

It supports global and local variable declarations, a single function definition (e.g., int main() { … }), and statements including expression statements, if–else, while, and return. It also builds an advanced symbol table for variables.

The parser emits opcodes for a stack-based virtual machine.
